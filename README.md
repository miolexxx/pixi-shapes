#### PIXI SHAPES

## Usage

# Download
```bash
git clone git@github.com:miolex/pixi-shapes.git
```
# Build
```bash
cd pixi-shapes
npm install
npm run build
```
# Run server
```bash
cd ./dist/
http-server
```
